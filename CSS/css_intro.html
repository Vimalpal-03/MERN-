Cascading Style Sheets (CSS)

A technical guide to styling the web.

1. What is CSS?

CSS is the language used to style the visual presentation of a web page. While HTML provides the structure (the bones), CSS provides the presentation (the skin, hair, and clothes).

2. Syntax Anatomy

A CSS rule-set consists of a selector and a declaration block:

h1 {
    color: blue;
    font-size: 12px;
}
  • Selector: Points to the HTML element.
  • Declaration: A specific style rule (Property + Value).

3. Implementation Methods

Type Location Usage
Inline Inside HTML tags Specific overrides
Internal Inside <style> tags Single page styles
External Separate .css file Site-wide styling (Best Practice)

4. The Box Model

Everything in CSS is a box. The Box Model determines how much space an element takes up and how it interacts with other elements.

  • Content: Text or images.
  • Padding: Space inside the border.
  • Border: The frame around the padding.
  • Margin: Space outside the border.
