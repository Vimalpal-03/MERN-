Introduction to Flexbox

The Flexible Box Layout Module (Flexbox) provides a more efficient way to lay out, align, and distribute space among items in a container, even when their size is unknown or dynamic.

Key Concept: Flexbox is a 1-dimensional layout model. It deals with layout in either a row or a column at a time, but not both simultaneously (that's what CSS Grid is for).

To start using Flexbox, you must first define a flex container using display: flex;. Direct children of this container automatically become "flex items".

.container {
    display: flex;
}

gap

The gap property explicitly controls the space between flex items. It applies that spacing only between items, not on the outer edges.

.container {
    display: flex;
    gap: 20px; /* Applies to both rows and columns */
    /* row-gap: 10px; */
    /* column-gap: 30px; */
}

Flex Item Properties (Applied to Children)

While the above properties are applied to the parent .container, these properties are applied to the child .item elements themselves.

order

By default, flex items are laid out in the source order. The order property controls the order in which they appear.

.item-2 { order: -1; /* Moves this item to the front */ }

flex-grow

Defines the ability for a flex item to grow if necessary. It dictates what amount of the available space inside the flex container the item should take up.

Item 2 has `flex-grow: 1`

align-self

Allows the default alignment (or the one specified by align-items) to be overridden for individual flex items.

.special-item {
    align-self: flex-end; /* Pushes just this item to the bottom */
}
