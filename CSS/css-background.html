Shorthand

  • The background Property

1. background-color

Sets the background color of an element. The background color renders underneath background images.

.box {
    background-color: #be123c;       /* HEX Code */
    background-color: rgb(190, 18, 60);  /* RGB */
    background-color: rgba(0, 0, 0, 0.5); /* Semi-transparent */
}

2. background-image

Applies one or more background images to an element. Multiple images are separated by commas, with the first image appearing "on top" of the others. Gradients are also treated as background images.

linear-gradient()
url() Pattern

3. background-repeat

Defines how background images are tiled if they are smaller than the element.

  • repeat (Default): Tiles both X and Y axes.
  • repeat-x: Tiles only horizontally.
  • repeat-y: Tiles only vertically.
  • no-repeat: Does not tile.
  • space / round: Advanced distribution of tiles.
repeat-x
no-repeat

4. background-position

Sets the starting point of a background image. Accepts keywords (top, right, bottom, left, center), percentages, or pixel lengths.

.hero {
    /* x-axis y-axis */
    background-position: center top;
    
    /* Offsets: 10px from right edge, 20px from bottom edge */
    background-position: right 10px bottom 20px; 
}

5. background-size

Controls the scaling of the background image. Crucial for making backgrounds responsive.

contain (Preserves ratio, fits inside)
cover (Preserves ratio, fills box)

6. background-attachment

Determines if the background image scrolls with the page, or remains fixed to the viewport.

scroll (Moves with content)

local (Sticks to element content)

7. background-clip

Determines how far the background (color or image) should extend within the CSS Box Model.

  • border-box (Default): Extends to the outside edge of the border.
  • padding-box: Extends to the outside edge of the padding.
  • content-box: Clipped strictly to the content area.
  • text: The background paints only where the text characters are (requires transparent text color).
-webkit-background-clip: text;

CLIPPED TEXT

8. background-blend-mode

Defines how an element's background images should blend with each other and with the element's background color. (Works similarly to Photoshop blend modes).

.blend {
    background-image: url("photo.jpg");
    background-color: #be123c;
    background-blend-mode: multiply; /* screen, overlay, darken, etc. */
}

9. The background Shorthand

You can combine all these properties into a single background declaration. If background-size is included, it must immediately follow background-position, separated by a forward slash /.

Order: color image position/size repeat attachment

.card {
    /* Sets color, image, centers it, makes it cover, prevents repeat, and fixes it */
    background: #0f172a url("pattern.svg") center/cover no-repeat fixed;
}
