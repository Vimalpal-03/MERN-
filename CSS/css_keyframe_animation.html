CSS Keyframe Animations

The @keyframes CSS at-rule controls the intermediate steps in a CSS animation sequence by defining styles for keyframes (or waypoints) along the animation sequence. This gives more control over the intermediate steps of the animation sequence than transitions.

1. Continuous Pulse Effect

This animation scales an element up slightly and reduces its opacity, then returns it to its original state, creating a breathing or pulsing effect.

/* Define the keyframes */
@keyframes pulse-anim {
    0% { transform: scale(1); opacity: 1; }
    50% { transform: scale(1.1); opacity: 0.7; }
    100% { transform: scale(1); opacity: 1; }
}

/* Apply to the element */
.element {
    animation: pulse-anim 2s ease-in-out infinite;
}

2. Loading Spinner

A classic loading spinner created by rotating an element with a partial border 360 degrees infinitely.

/* Define the keyframes */
@keyframes spin-anim {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}

/* Apply to the element */
.spinner {
    border: 6px solid #d1d5db;
    border-top-color: #3b82f6;
    border-radius: 50%;
    animation: spin-anim 1s linear infinite;
}

3. Bouncing Ball

By manipulating the translateY property and adjusting the cubic-bezier timing functions midway through the animation, we can simulate realistic gravity and bouncing.

/* Define the keyframes */
@keyframes bounce-anim {
    0%, 100% { 
        transform: translateY(0); 
        animation-timing-function: cubic-bezier(0.8, 0, 1, 1); 
    }
    50% { 
        transform: translateY(-50px); 
        animation-timing-function: cubic-bezier(0, 0, 0.2, 1); 
    }
}

/* Apply to the element */
.ball {
    border-radius: 50%;
    animation: bounce-anim 1s infinite;
}
