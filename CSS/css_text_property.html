Introduction

While font properties handle the typeface and size of your characters, text properties control the visual formatting of the block of text. This includes how it aligns, spaces out, wraps, and renders decorative elements like lines and shadows.

Text Alignment

text-align

The text-align property is used to set the horizontal alignment of a text within its container.

Value Description
left Aligns text to the left edge (default for LTR languages).
right Aligns text to the right edge.
center Centers the text.
justify Stretches the lines so that each line has equal width. 
h1 {
    text-align: center;
}
p.newspaper {
    text-align: justify;
}

Text Decoration

text-decoration

The text-decoration property is a shorthand for applying lines to text (like underlines or strikethroughs). It combines line, color, style, and thickness.

/* Basic underline */
a {
    text-decoration: underline;
}

/* Advanced: Line, Style, Color, Thickness */
h2 {
    text-decoration: underline wavy red 2px;
}

/* Removing decoration */
.no-line {
    text-decoration: none;
}

This text has a red, wavy underline.

Text Transformation

text-transform

The text-transform property controls the capitalization of text, allowing you to change case without altering the HTML source code.

Value Result Example
uppercase THIS IS UPPERCASE
lowercase this is lowercase
capitalize This Is Capitalized

Text Spacing & Indentation

text-indent

Used to specify the indentation of the first line of a text block.

p {
    text-indent: 50px;
}

letter-spacing & word-spacing

These properties adjust the empty space between characters and whole words, respectively.

.wide-letters {
    letter-spacing: 3px;
}
.wide-words {
    word-spacing: 10px;
}

Text Shadow

text-shadow

The text-shadow property applies one or more shadows to text. It takes four values: horizontal offset, vertical offset, blur radius, and color.

/* offset-x | offset-y | blur-radius | color */
h1 {
    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
}

/* Multiple shadows */
.neon {
    text-shadow: 0 0 5px #fff, 0 0 10px #0ea5e9;
}

Wrapping & Breaking

white-space

Controls how whitespace and line breaks inside an element are handled.

  • nowrap: Text will never wrap to the next line. It continues on the same line until a <br> tag is met.
  • pre: Whitespace is preserved exactly as written in the HTML (like a <pre> tag).

word-break & overflow-wrap

These properties determine what happens when a single word is too long to fit in its container.

.long-url {
    /* Breaks a long word exactly at the container boundary */
    word-break: break-all; 
    
    /* Better alternative: only breaks if it absolutely has to */
    overflow-wrap: break-word; 
}
