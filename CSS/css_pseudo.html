CSS Pseudo-classes & Pseudo-elements

Welcome to the documentation. These advanced CSS selectors allow you to style elements based on their state, their position in the document tree, or even create entirely new virtual elements without adding extra HTML markup.

1. Pseudo-classes

A pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s). They are prefixed with a single colon :.

Interactive States (:hover, :active, :focus)

These are triggered by user interaction. Try hovering, clicking, and pressing the Tab key to focus the button below.

.btn-demo:hover {
  background-color: #4f46e5;
  transform: translateY(-2px);
}

.btn-demo:active {
  transform: translateY(0);
}

.btn-demo:focus {
  outline: 3px solid #a5b4fc;
}

Structural Selectors (:nth-child)

Target elements based on their position among siblings. Highly useful for alternating styles in tables or lists.

  • Item 1 (Odd)
  • Item 2 (Even)
  • Item 3 (Odd)
  • Item 4 (Even)
/* Styles every odd list item */
.list-demo li:nth-child(odd) {
  background-color: #f8fafc;
}

/* Styles every even list item */
.list-demo li:nth-child(even) {
  background-color: #ffffff;
}

/* You can combine them! */
.list-demo li:hover {
  background-color: #e0e7ff;
}

2. Pseudo-elements

A pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element. In CSS3, they are prefixed with a double colon :: to distinguish them from pseudo-classes.

Generating Content (::before, ::after)

These allow you to insert content onto a page from CSS. The content property is strictly required for these to render.

CSS pseudo-elements let you add decorative flair and icons without cluttering up your clean HTML structure. 
.quote-demo {
  position: relative;
  padding-left: 3rem;
}

/* Injects a giant quote mark inside the element, before the text */
.quote-demo::before {
  content: "❝";
  position: absolute;
  top: 10px;
  left: 10px;
  font-size: 3rem;
  color: #cbd5e1;
}

Typography Control (::first-letter, ::selection)

Highlight the paragraph below with your mouse cursor to see the custom selection color. Notice the magazine-style drop cap created automatically.

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML. Select this text to see the custom highlight color in action!

/* Creates a classic drop-cap effect */
.typography-demo::first-letter {
  float: left;
  font-size: 4rem;
  font-weight: 900;
  color: #6366f1;
}

/* Overrides the browser's default blue highlight color */
.typography-demo::selection {
  background-color: #fce7f3;
  color: #be185d;
}
