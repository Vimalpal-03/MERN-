Cascading Style Sheets: Selectors

This document provides a complete reference to CSS selectors, categorized by their structural types.

1. Simple Selectors

These target elements based on their core identity (name, id, or class).

Type (Element) Selector

Selects all elements of a specific HTML tag.

p {
  color: #333;
}

Class Selector

Selects all elements containing a specific class attribute. Prefixed with a period (.).

.alert-box {
  border: 1px solid red;
}

ID Selector

Selects a single, unique element based on its ID attribute. Prefixed with a hash (#).

#main-header {
  font-weight: bold;
}

Universal Selector

Selects all elements within the document. Represented by an asterisk (*).

* {
  box-sizing: border-box;
}

2. Attribute Selectors

These target elements based on the presence or value of HTML attributes.

Syntax Description
[attr] Has the specified attribute.
[attr="value"] Attribute has the exact value.
[attr~="value"] Attribute contains the value as a whole word in a space-separated list.
[attr^="value"] Attribute value begins with the specified string.
[attr$="value"] Attribute value ends with the specified string.
[attr*="value"] Attribute value contains the specified substring anywhere. 
input[type="text"] {
  width: 100%;
}

3. Combinators

These establish relationships between two or more selectors.

Descendant Combinator (Space)

Selects nodes that are descendants (children, grandchildren, etc.) of the first element.

article p { margin-bottom: 1rem; }

Child Combinator (>)

Selects nodes that are direct children of the first element.

ul > li { list-style-type: square; }

Adjacent Sibling Combinator (+)

Selects the element that immediately follows the first element (must share the same parent).

h2 + p { margin-top: 0; }

General Sibling Combinator (~)

Selects all elements that follow the first element and share the same parent.

h2 ~ p { color: gray; }

4. Pseudo-classes

Keywords added to selectors that specify a special state of the targeted element.

Note: Pseudo-classes always begin with a single colon (:).
  • :hover - Triggers when the user points at an element.
  • :focus - Triggers when an element receives input focus.
  • :nth-child(n) - Targets an element based on its position in a list of siblings.
  • :first-child / :last-child - Targets the first or last element among siblings.
  • :not(selector) - Targets elements that do not match the provided selector.
a:hover {
  text-decoration: underline;
}

li:nth-child(even) {
  background-color: #f2f2f2;
}

5. Pseudo-elements

Keywords added to selectors that let you style a specific part of the selected element.

Note: In CSS3, pseudo-elements use a double colon (::) to distinguish them from pseudo-classes.
  • ::before - Inserts generated content before the element's actual content.
  • ::after - Inserts generated content after the element's actual content.
  • ::first-letter - Styles the first letter of a block-level element.
  • ::first-line - Styles the first line of text in a block-level element.
  • ::selection - Styles the portion of a document that has been highlighted by the user.
p::first-letter {
  font-size: 2em;
  font-weight: bold;
}

.quote::before {
  content: "“";
  color: gray;
}
