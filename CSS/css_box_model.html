The CSS Box Model

When laying out a document, the browser's rendering engine represents each element as a rectangular box according to the standard CSS basic box model. The box model consists of four distinct areas from the inside out: Content, Padding, Border, and Margin.

Width × Height
  • Content edge: The innermost area containing the actual content (text, images, or other media).
  • Padding edge: Clears an area around the content. The padding is transparent, but takes on the element's background color.
  • Border edge: Wraps around the padding and content.
  • Margin edge: Clears an area outside the border. The margin is completely transparent and separates the element from its neighbors.

Box Sizing

The box-sizing property dictates how the total width and height of an element are calculated.

Best Practice: Modern web development almost exclusively relies on box-sizing: border-box; to make sizing more predictable.

content-box (Default)

When you set width and height, you are setting the dimensions of the content area only. Any padding or borders are added to the outside of that width.

border-box

When you set width and height, that dimension includes the content, padding, and border. If you set a width of 300px, the total rendered width will be exactly 300px, and the content area will shrink to accommodate the padding and borders.

/* Universal Box-Sizing Reset */
*, *::before, *::after {
    box-sizing: border-box;
}

.card {
    width: 300px;
    padding: 20px;
    border: 2px solid black;
    /* Because of border-box, the rendered width is exactly 300px */
}

Width & Height Constraints

These properties control the size of the content area (or the entire box if border-box is used).

  • width / height: Sets explicit dimensions.
  • max-width / max-height: Prevents an element from exceeding a certain size, allowing it to shrink responsively.
  • min-width / min-height: Ensures an element is at least a certain size, but allows it to grow if content overflows.
.container {
    width: 100%;
    max-width: 1200px; /* Responsive maximum limit */
    min-height: 100vh; /* Fills at least the entire viewport height */
}

Padding

The padding property creates space inside the element, between the content and the border.

.alert {
    /* Shorthand 4 values: Top | Right | Bottom | Left (Clockwise) */
    padding: 10px 20px 15px 5px; 

    /* Shorthand 2 values: Vertical (Top/Bottom) | Horizontal (Left/Right) */
    padding: 2rem 4rem; 
    
    /* 1 value: Applies equally to all four sides */
    padding: 20px; 
    
    /* Individual properties */
    padding-top: 10px;
    padding-left: 20px;
}

Border

The border property allows you to specify the style, width, and color of an element's border. A border will not render unless a border-style is provided.

.button {
    /* Shorthand: width | style | color */
    border: 2px solid #2563eb;
    
    /* Rounding the corners of the border */
    border-radius: 8px; 
}

.divider {
    /* Applying border to a single side */
    border-bottom: 1px dashed #cbd5e1;
}

Margin

The margin property creates completely transparent space outside the border edge, pushing adjacent elements away.

Margin Collapsing: When the vertical margins (top and bottom) of two block-level elements touch, they will combine (collapse) into a single margin whose size is the larger of the two margins. 
.block-element {
    /* Often used to center a block element horizontally */
    margin: 0 auto; 
    
    /* Top | Right | Bottom | Left */
    margin: 20px 0 40px 0;
    
    /* Margins can accept negative values to overlap elements */
    margin-top: -10px; 
}

Outline

The outline property draws a line outside the border edge. Unlike borders, outlines do not take up space in the box model. Adding an outline will not cause layout reflows or push other elements around.

Outlines are most commonly used for accessibility to indicate keyboard focus states.

input:focus {
    /* width | style | color */
    outline: 3px solid #2563eb;
    
    /* Creates a transparent gap between the border and the outline */
    outline-offset: 2px; 
}
