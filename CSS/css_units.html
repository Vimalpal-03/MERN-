Introduction

CSS has several different units for expressing a length. Many CSS properties take "length" values, such as width, margin, padding, font-size, etc.

Length is a number followed by a length unit, such as 10px, 2em, etc. A whitespace cannot appear between the number and the unit. However, if the value is 0, the unit can be omitted.

Absolute Units

The absolute length units are fixed and a length expressed in any of these will appear as exactly that size. They are not recommended for use on screen, because screen sizes vary so much. However, they can be used if the output medium is known, such as for print layout.

Unit Description Equivalent to
px Pixels (1px = 1/96th of 1in) Commonly used for precise screen measurements.
cm Centimeters 96px/2.54
mm Millimeters 1/10th of 1cm
in Inches 96px
pt Points 1/72 of 1in
pc Picas 12pt

Relative Units

Relative length units specify a length relative to another length property. Relative length units scale better between different rendering mediums.

Unit Description
em Relative to the font-size of the element (2em means 2 times the size of the current font).
rem Relative to font-size of the root element (usually the <html> tag). Ideal for consistent typography.
% Relative to the parent element. Often used for fluid widths and layouts.
ch Relative to width of the "0" (zero) character of the element's font.

Viewport-Percentage Lengths

The viewport units are relative to the size of the viewport (the browser window size). If the viewport changes size, the elements size using these units will automatically scale.

Unit Description
vw Relative to 1% of the width of the viewport.
vh Relative to 1% of the height of the viewport.
vmin Relative to 1% of viewport's smaller dimension.
vmax Relative to 1% of viewport's larger dimension.

Usage Examples

Here is an example showing how different units are applied in standard CSS:

/* Absolute Unit Example */
.box {
    width: 300px;
    border: 2px solid black;
}

/* Relative Unit Example (rem vs em) */
html {
    font-size: 16px; /* Base size */
}

.parent {
    font-size: 1.5rem; /* 24px */
}

.child {
    /* 2em here is relative to parent's 24px, so this is 48px */
    font-size: 2em; 
}

/* Viewport Unit Example */
.hero-section {
    width: 100vw;   /* Full viewport width */
    height: 100vh;  /* Full viewport height */
    padding: 5%;    /* 5% of parent container's width */
}
