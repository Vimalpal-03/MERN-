CSS Media Queries

Media queries are a fundamental part of responsive web design. They allow you to apply CSS styles only when a specific condition is met, such as the viewport width being larger or smaller than a certain size, or if the device is a screen rather than printed paper.

Note: Media queries were introduced in CSS3 and are supported by all modern browsers.

Basic Syntax

A media query consists of a media type and can contain one or more expressions, which resolve to either true or false.

@media not|only mediatype and (mediafeature and|or|not mediafeature) {
  /* CSS rules go here */
}

If the specified media type matches the type of device the document is being displayed on and all expressions in the media query are true, the CSS rules are applied.

Media Types

There are several media types, but the most commonly used ones are:

  • all - Used for all media type devices (Default).
  • print - Used for printers.
  • screen - Used for computer screens, tablets, smart-phones etc.
  • speech - Used for screenreaders that "read" the page out loud.

Example: Print Styling

You can hide elements when printing a webpage:

@media print {
  .navigation-bar, .footer {
    display: none;
  }
}

Media Features

Media features describe specific characteristics of the user agent, output device, or environment. The most common features revolve around viewport dimensions:

  • max-width: Rules apply if the viewport width is less than or equal to the value.
  • min-width: Rules apply if the viewport width is greater than or equal to the value.
  • orientation: Checks if the device is in landscape or portrait mode.

Common Breakpoints (Examples)

Here is a standard set of media queries using mobile-first design principles (using min-width):

/* Extra small devices (phones, 600px and down) */
/* Default CSS goes outside media queries */

/* Small devices (portrait tablets and large phones, 600px and up) */
@media only screen and (min-width: 600px) {
  body {
    background-color: lightblue;
  }
}

/* Medium devices (landscape tablets, 768px and up) */
@media only screen and (min-width: 768px) {
  body {
    background-color: lightgreen;
  }
}

/* Large devices (laptops/desktops, 992px and up) */
@media only screen and (min-width: 992px) {
  body {
    background-color: lightcoral;
  }
}

/* Extra large devices (large laptops and desktops, 1200px and up) */
@media only screen and (min-width: 1200px) {
  body {
    background-color: lightgoldenrodyellow;
  }
}
