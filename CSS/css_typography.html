Introduction

CSS provides a wide range of properties to style text and fonts. These properties govern everything from the typeface used to the exact spacing between letters and words, allowing for precise typographic control on the web.

Font Properties

1. font-family

Specifies the typeface to be used. It's best practice to provide a "fallback" list of fonts, ending with a generic family (like serif or sans-serif) in case the user doesn't have the first font installed.

p {
    font-family: "Helvetica Neue", Helvetica, Arial, sans-serif;
}

2. font-size

Sets the size of the text. Common units include px (pixels), em (relative to parent), and rem (relative to the root HTML element).

h1 {
    font-size: 2.5rem;
}

3. font-weight

Sets the thickness or boldness of the font. It accepts keywords or numeric values (from 100 to 900).

Value Description
normal Standard font weight (equivalent to 400).
bold Thick font weight (equivalent to 700).
100 - 900 Numeric weights. 100 is thin, 400 is normal, 700 is bold, 900 is black/heavy.

Text Formatting

4. color

Sets the color of the text. You can use named colors, HEX codes, RGB, or HSL values.

.text-primary {
    color: #005b9f; /* HEX */
    /* color: rgb(0, 91, 159); */
    /* color: blue; */
}

5. text-align

Sets the horizontal alignment of the text inside its container.

  • left: Aligns to the left edge (default for LTR languages).
  • right: Aligns to the right edge.
  • center: Centers the text.
  • justify: Stretches lines so that every line has equal width (like a newspaper).

6. text-decoration

Used to set or remove decorations added to text. Most commonly used to remove underlines from links. 

a {
    text-decoration: none; /* Removes underline */
}
.strike {
    text-decoration: line-through;
}

7. text-transform

Controls the capitalization of text. Useful for styling headings without changing the actual HTML content.

  • uppercase: TRANSFORMS ALL LETTERS TO CAPITALS.
  • lowercase: transforms all letters to lowercase.
  • capitalize: Transforms The First Letter Of Each Word.
  • none: Leaves the text as typed.

Spacing Properties

8. letter-spacing

Increases or decreases the space between characters in a text.

h2 {
    letter-spacing: 2px; /* Spreads letters apart */
}

9. word-spacing

Increases or decreases the space between whole words in a text.

p {
    word-spacing: 0.5em; /* Increases space between words */
}

10. line-height

Specifies the height of a line of text (leading). Using a unitless number is the recommended best practice, as it acts as a multiplier of the element's font size.

body {
    line-height: 1.6; /* 1.6 times the current font size */
}

Combined Example

Here is how these properties look when used together to style a block of text.

.styled-article {
    font-family: "Georgia", serif;
    font-size: 1.125rem;
    font-weight: 400;
    color: #333333;
    text-align: justify;
    line-height: 1.8;
    word-spacing: 0.1em;
}

.styled-article h3 {
    font-family: "Arial", sans-serif;
    font-weight: 700;
    text-transform: uppercase;
    letter-spacing: 1px;
    text-decoration: underline;
    color: #005b9f;
}

Typography Matters

Good typography is essential for web design. By carefully adjusting the font family, line height, and spacing, we make text much easier for users to read. This paragraph demonstrates a justified alignment with an increased line height and slight word spacing applied to a serif font.
