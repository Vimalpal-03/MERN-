Introduction

The CSS filter property provides access to graphical effects like blurring, color shifting, or contrast adjustments. These effects are applied to an element before it is drawn to the screen. It accepts one or more <filter-function> values.

.element {
    filter: blur(5px) grayscale(50%);
}

blur()

Applies a Gaussian blur. The passed parameter determines the radius of the blur. It accepts length values (like px, em) but does not accept percentages.

brightness()

Adjusts the visual brightness of the element. 0% or 0 creates a completely black element. 100% or 1 leaves the image unchanged. Values over 100% make the image brighter than the original.

contrast()

Adjusts the contrast of the input. 0% makes the element completely gray. 100% is unchanged. Values over 100% increase the contrast, making lights lighter and darks darker.

drop-shadow()

Applies a drop shadow. Unlike box-shadow (which draws a rigid rectangle around the bounding box), drop-shadow() respects transparent alpha channels, meaning the shadow conforms to the actual visual shape of the element (perfect for transparent PNGs or SVGs).

Syntax: drop-shadow(offset-x offset-y blur-radius color)

Hover to apply: filter: drop-shadow(10px 10px 5px rgba(0,0,0,0.6));

grayscale()

Converts the element's colors to grayscale. 100% is completely black and white. 0% leaves the element unchanged.

hue-rotate()

Applies a hue rotation based on the color circle. The value is specified in degrees (deg). A value of 360deg is a full rotation, returning the element to its original colors.

Combining Multiple Filters

You can chain multiple filter functions together by separating them with a space. The browser will apply them in the exact order they are written, which can drastically change the final outcome.

.complex-effect {
    /* Desaturates slightly, shifts hue, then adds a shadow */
    filter: saturate(50%) hue-rotate(45deg) drop-shadow(4px 4px 10px #000);
}
