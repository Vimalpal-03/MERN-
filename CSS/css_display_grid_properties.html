1. Core Terminology

Grid Cell

A Grid Cell is the smallest unit on a grid. It is the space between four intersecting grid lines, much like a single table cell.

Grid Track

A Grid Track is the space between two adjacent grid lines. You can think of them as the vertical columns or horizontal rows of your grid.

2. Container Properties

Properties applied directly to the parent element carrying display: grid;.

grid-template-columns & grid-template-rows

These properties define the size and number of grid tracks (rows and columns).

.container {
  display: grid;
  grid-template-columns: 100px 1fr 2fr;
  grid-template-rows: 60px auto;
}

gap

The gap property dictates the size of the gutters between grid rows and columns.

justify-content & align-content

When the total size of your grid is strictly smaller than the size of its grid container, you can align the entire grid inside the container.

  • justify-content: Aligns the grid along the inline (horizontal) axis.
  • align-content: Aligns the grid along the block (vertical) axis. If set to stretch (default), rows stretch to fill the container height.

align-items

While align-content aligns the tracks, align-items aligns the content inside the individual grid cells along the vertical axis.

.container-align {
  height: 200px;
  justify-content: space-between; /* Horizontal spread */
  align-content: stretch;         /* Vertical track stretching */
  align-items: center;            /* Vertical item centering */
}

Notice how tracks spread horizontally, stretch vertically, but items are centered inside their cells.

3. Child (Grid Item) Properties

Properties applied directly to the direct children of a grid container.

grid-column-start/end & grid-row-start/end

These determine a grid item's location and size by referring to specific grid lines. You can also use the shorthand grid-column and grid-row.

grid-area

A shorthand for grid-row-start / grid-column-start / grid-row-end / grid-column-end.

.item-a {
  grid-column-start: 1;
  grid-column-end: 3;
  grid-row-start: 1;
  grid-row-end: 3;
}
.item-b {
  /* row-start / col-start / row-end / col-end */
  grid-area: 1 / 3 / 2 / 5;
}

justify-self & align-self

Overrides the container's align-items and justify-items for a specific, single grid item.

  • justify-self: Aligns the item horizontally inside its own cell.
  • align-self: Aligns the item vertically inside its own cell.
.child-item {
  justify-self: end; /* Pushed to the right edge of cell */
  align-self: end;   /* Pushed to the bottom edge of cell */
}
